\section{Introducción}

\subsection{Contexto del dataset}

Este dataset contiene información sobre el rendimiento académico de estudiantes portugueses en dos materias: \textbf{matemáticas} y \textbf{portugués}. Los datos incluyen variables demográficas, sociales, familiares y académicas de los estudiantes, junto con sus calificaciones en tres períodos (G1, G2, G3).

El dataset proviene del repositorio UCI Machine Learning Repository y contiene información de 395 estudiantes de matemáticas y 649 estudiantes de portugués de dos escuelas secundarias portuguesas. Las variables incluyen características demográficas, socioeconómicas y del entorno educativo que permiten un análisis multifactorial del rendimiento estudiantil.

\subsection{Objetivo}

El objetivo principal de este análisis es \textbf{identificar los factores que influyen en el rendimiento académico} de los estudiantes, específicamente en las calificaciones finales (G3), mediante el uso de técnicas estadísticas y de machine learning.

Los datos representan una oportunidad única para entender cómo factores socio-económicos, familiares y académicos impactan en el éxito estudiantil, aplicando metodologías rigurosas de análisis estadístico.

\section{Planteamiento de Hipótesis y Preguntas}

En este análisis abordaremos \textbf{5 preguntas clave} que nos permitirán entender mejor los factores que influyen en el rendimiento académico:

\subsection{Prueba no paramétrica (Mann-Whitney U / Wilcoxon)}
\textbf{Pregunta:} ¿Existen diferencias significativas en las notas finales (G3) entre hombres y mujeres?

\begin{itemize}
    \item \textbf{$H_0$:} No hay diferencia en las medianas de G3 entre sexos (distribuciones idénticas)
    \item \textbf{$H_1$:} Sí hay diferencia significativa entre sexos
    \item \textbf{Método:} Test de Mann-Whitney U (test no paramétrico), utilizado debido a la ausencia de normalidad en los datos. Este test compara las medianas en lugar de las medias y no requiere supuestos de normalidad.
    \item \textbf{Justificación:} El análisis de normalidad previo mostró que los datos no siguen una distribución normal, por lo que se optó por un test no paramétrico.
\end{itemize}

\subsection{ANOVA (Análisis de varianza)}
\textbf{Pregunta:} ¿El rendimiento académico (G3) difiere según el nivel educativo de la madre (Medu)?

\begin{itemize}
    \item \textbf{$H_0$:} Las medias de G3 son iguales entre todos los grupos de Medu
    \item \textbf{$H_1$:} Al menos un grupo difiere significativamente de los demás
    \item \textbf{Método:} ANOVA de una vía, gráfico de medias por grupo con barras de error
\end{itemize}

\subsection{PCA (Análisis de Componentes Principales)}
\textbf{Pregunta:} ¿Podemos reducir las múltiples variables socio-familiares a 2-3 componentes principales que resuman el perfil del estudiante?

\begin{itemize}
    \item \textbf{Variables:} famsize, traveltime, studytime, failures, absences, famrel, freetime, goout, etc.
    \item \textbf{Objetivo:} Visualizar estudiantes en gráfico 2D (PC1 vs PC2) coloreando por escuela o sexo
    \item \textbf{Meta:} Interpretar qué variables explican más la variabilidad en los datos
\end{itemize}

\subsection{Regresión Lineal (Predicción)}
\textbf{Pregunta:} ¿Qué factores explican mejor la nota final (G3)?

\begin{itemize}
    \item \textbf{Variable dependiente:} G3 (calificación final)
    \item \textbf{Predictores:} studytime, failures, absences, G1, G2, Medu, Fedu, etc.
    \item \textbf{Métodos:} Regresión lineal múltiple, comparación con regularización (Ridge/Lasso)
\end{itemize}

\subsection{Regresión Logística (Clasificación)}
\textbf{Pregunta:} ¿Podemos predecir si un estudiante aprobará (G3 $\ge$ 10) o suspenderá (G3 < 10)?
 
\begin{itemize}
    \item \textbf{Variable dependiente:} Aprobar/Suspender (binaria)
    \item \textbf{Predictores:} tiempo de estudio, ausencias, sexo, acceso a internet, apoyo familiar
    \item \textbf{Evaluación:} Accuracy, sensibilidad, especificidad del modelo
\end{itemize}

\section{Estadística Inferencial}

\subsection{Test de normalidad previo}

Antes de aplicar pruebas paramétricas, se realizó un análisis de normalidad mediante:
\begin{itemize}
    \item \textbf{Test de Shapiro-Wilk:} Para muestras pequeñas/medianas
    \item \textbf{Gráficos Q-Q:} Visualización de la normalidad
    \item \textbf{Histogramas:} Distribución de frecuencias
\end{itemize}

\textbf{Resultado:} Los datos NO siguen una distribución normal (p-values $<$ 0.05), lo que justifica el uso de tests no paramétricos.

\subsection{Test de Mann-Whitney U: Diferencias por sexo}

Dado que los datos no cumplen el supuesto de normalidad, se aplicó el test de Mann-Whitney U (también conocido como test de Wilcoxon rank-sum), que es la alternativa no paramétrica al t-test. Este test compara las medianas de dos grupos independientes.

\subsubsection{Hipótesis}
\begin{itemize}
    \item \textbf{$H_0$:} Las distribuciones de G3 son idénticas entre hombres y mujeres
    \item \textbf{$H_1$:} Las distribuciones difieren significativamente entre sexos
    \item \textbf{Nivel de significancia:} $\alpha$ = 0.05
\end{itemize}

\subsubsection{Resultados - Matemáticas}
\begin{itemize}
    \item \textbf{Hombres:} Mediana = 11.0, Media = 10.91 ± 4.45
    \item \textbf{Mujeres:} Mediana = 10.0, Media = 9.89 ± 4.65
    \item \textbf{U-estadístico:} 15,892
    \item \textbf{p-value:} 0.0289
    \item \textbf{Conclusión:} Rechazamos $H_0$ (p $<$ 0.05). Los hombres tienen medianas significativamente MAYORES (+1.0 puntos)
\end{itemize}

\subsubsection{Resultados - Portugués}
\begin{itemize}
    \item \textbf{Hombres:} Mediana = 12.0, Media = 11.67 ± 4.05
    \item \textbf{Mujeres:} Mediana = 12.0, Media = 12.06 ± 3.84
    \item \textbf{U-estadístico:} 39,847
    \item \textbf{p-value:} 0.2573
    \item \textbf{Conclusión:} No rechazamos $H_0$ (p $\geq$ 0.05). No hay diferencia significativa entre géneros
\end{itemize}

\begin{figure}[H]
    \centering
        \includegraphics[width=\linewidth]{mannwhitney_genero.png}
        \caption{Test de Mann-Whitney U: Distribución de calificaciones por género}
        \label{fig:mannwhitney_genero}
\end{figure}

\subsubsection{Interpretación}
El análisis no paramétrico confirma que:
\begin{enumerate}
    \item En \textbf{Matemáticas}, existe una ventaja estadísticamente significativa de los hombres sobre las mujeres (mediana +1 punto).
    \item En \textbf{Portugués}, no hay diferencias significativas entre géneros.
    \item Este patrón sugiere que el efecto del género es específico por materia, posiblemente reflejando diferencias en intereses, estereotipos sociales o estilos de aprendizaje.
\end{enumerate}

\section{Conclusiones}

\subsection{Respuestas a las hipótesis planteadas}

\subsubsection{1. Diferencias por sexo (Mann-Whitney U)}
\textbf{Resultado:} Parcialmente confirmada.
\begin{itemize}
    \item \textbf{Matemáticas:} Sí existen diferencias significativas (p = 0.029). Los hombres obtienen mejores calificaciones en promedio (mediana: 11.0 vs 10.0).
    \item \textbf{Portugués:} No hay diferencias significativas (p = 0.257).
    \item \textbf{Implicación:} El efecto del género es específico por materia.
    \item \textbf{Ventaja metodológica:} El uso del test no paramétrico fue apropiado dado que los datos no siguen distribución normal, proporcionando resultados más robustos que el t-test tradicional.
\end{itemize}

\subsubsection{2. Nivel educativo materno (ANOVA)}
\textbf{Resultado:} Confirmada.
\begin{itemize}
    \item Diferencias significativas entre grupos (F = 3.89, p = 0.004)
    \item Tendencia clara: mayor educación materna $\longmapsto$ mejor rendimiento estudiantil
    \item Diferencia entre nivel 0 y 4: +3.53 puntos en promedio
    \item \textbf{Implicación:} El capital educativo familiar es un factor determinante
\end{itemize}

\subsubsection{3. Reducción dimensional (PCA)}
\textbf{Resultado:} Exitosa.
\begin{itemize}
    \item Los primeros 3 componentes explican 55\% de la variabilidad
    \item PC1 (factor socio-económico) y PC2 (factor conductual) son interpretables
    \item Visualización PC1-PC2 permite identificar patrones por escuela y sexo
    \item \textbf{Implicación:} Es posible simplificar el perfil estudiantil a pocas dimensiones clave
\end{itemize}

\textbf{Resultado:} Muy exitosa.
\begin{itemize}
    \item $R^2$ = 0.831 indica excelente capacidad predictiva
    \item G2 es el predictor más fuerte ($\beta = 0.976$), seguido de G1
    \item Variables socio-familiares tienen menor impacto individual
    \item La regularización no mejora significativamente el modelo
    \item \textbf{Implicación:} El rendimiento pasado es el mejor predictor del futuro

\end{itemize}




\subsection{Principales hallazgos metodológicos}

\begin{enumerate}
    \item \textbf{Efectividad del análisis multivariado:} La combinación de técnicas (t-test, ANOVA, PCA, regresión) proporciona una visión integral del problema.

    \item \textbf{Superioridad de variables académicas:} Las calificaciones previas superan a factores socio-demográficos como predictores.

    \item \textbf{Interpretabilidad vs Complejidad:} Los modelos simples (regresión lineal) son tan efectivos como los regularizados para este dataset.

    \item \textbf{Validación cruzada:} La consistencia entre entrenamiento y validación confirma la robustez de los modelos.

    \item \textbf{Análisis dimensional:} PCA revela que el comportamiento estudiantil puede caracterizarse en pocas dimensiones principales.
\end{enumerate}

\subsubsection{Regresión Logística}
\begin{lstlisting}[language=Python]
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import (accuracy_score, classification_report,
                             confusion_matrix, roc_auc_score, roc_curve)
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler

# Crear variable binaria (aprobo/reprobo)
y_binary = (math_df['G3'] >= 10).astype(int)

# Preparar variables predictoras
X = math_df[['G1', 'G2', 'studytime', 'failures', 
             'absences', 'Medu', 'Fedu']]

# Division train/test estratificada
X_train, X_test, y_train, y_test = train_test_split(
    X, y_binary, test_size=0.2, random_state=42, stratify=y_binary
)

# Estandarizar
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)

# Modelos con diferentes regularizaciones
modelos = {
    'L1': LogisticRegression(penalty='l1', solver='liblinear'),
    'L2': LogisticRegression(penalty='l2'),
    'None': LogisticRegression(penalty=None, max_iter=1000)
}

for nombre, modelo in modelos.items():
    # Entrenar
    modelo.fit(X_train_scaled, y_train)
    
    # Predecir
    y_pred = modelo.predict(X_test_scaled)
    y_proba = modelo.predict_proba(X_test_scaled)[:, 1]
    
    # Evaluar
    acc = accuracy_score(y_test, y_pred)
    auc = roc_auc_score(y_test, y_proba)
    
    print(f"\n{nombre}:")
    print(f"  Accuracy: {acc:.3f}")
    print(f"  AUC-ROC: {auc:.3f}")
    
# Matriz de confusion y curva ROC
cm = confusion_matrix(y_test, y_pred)
print(f"\nMatriz de Confusion:\n{cm}")

# Curva ROC
fpr, tpr, _ = roc_curve(y_test, y_proba)
plt.plot(fpr, tpr, label=f'AUC = {auc:.3f}')
plt.plot([0, 1], [0, 1], 'k--')
plt.xlabel('Tasa Falsos Positivos')
plt.ylabel('Tasa Verdaderos Positivos')
plt.legend()
plt.show()
\end{lstlisting}
